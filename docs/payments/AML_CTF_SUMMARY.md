# AML/CTF Summary (Partner-led) - Draft

**Effective Date:** [DATE]

## 1) Model
Handyman uses a licensed partner to hold funds and perform regulated AML/CTF monitoring and reporting. Handyman supports partner compliance by collecting required user information, enforcing flow controls, and providing audit data.

## 2) Platform controls that reduce risk
- identity verification steps for handymen (ID type/number/image)
- transaction history and audit logging
- restrictions on cancellations after job start
- evidence collection via before/after images

## 3) Red flags (examples)
- multiple accounts sharing the same identity details
- repeated failed verifications
- unusual top-up volumes for new accounts
- rapid repeated cancellations intended to move funds

## 4) Cooperation with partner
Handyman provides partner with:
- user identifiers and KYC artifacts (as permitted)
- transaction references
- dispute and evidence records on request
