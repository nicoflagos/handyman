# Cap Table Example — Pre-money ₦500m / Raise ₦15m (Draft)

**Date:** 2026-03-07  
**Note:** Illustrative only. Final cap table depends on issued shares and any ESOP.

## Inputs
- Pre-money valuation: ₦500,000,000
- Investment: ₦15,000,000
- Post-money: ₦515,000,000
- Investor ownership post-money: **2.91%**

## Founder impact (assuming founder owns 90% pre-round)
Ownership scales by factor = pre / post = 500 / 515 = **0.9708738**
- Founder post-round ≈ 90% × 0.9708738 = **87.38%**
- Other existing holders post-round ≈ 10% × 0.9708738 = **9.71%**
- New investor = **2.91%**

## Share math example (issued shares approach)
Assume:
- Pre-round issued shares: 10,000,000 ordinary shares

Then:
- Price/share = 500,000,000 / 10,000,000 = ₦50/share
- Investor shares = 15,000,000 / 50 = 300,000 shares
- Post-round shares = 10,300,000 shares
- Investor % = 300,000 / 10,300,000 = 2.91%

## ESOP timing effect (plain language)
- If ESOP is created **before** closing: founder dilution increases; investor tends to be protected.
- If ESOP is created **after** closing: founder + investor both dilute pro-rata.
