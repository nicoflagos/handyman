# Verification Appeal Process (Draft)

**Effective Date:** [DATE]

## 1) Purpose
Provide a fair way for handymen to appeal a rejected verification.

## 2) How to appeal
User submits:
- account ID
- verification attempt ID (if available)
- corrected information and new images

## 3) Review
- A different reviewer (not the one who rejected) handles the appeal.
- Decision within [X] business days.

## 4) Outcomes
- Approved (verified)
- Rejected with clear reason code and next steps
- Suspended if fraud is suspected
