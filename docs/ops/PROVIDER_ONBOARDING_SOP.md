# Provider (Handyman) Onboarding SOP (Draft)

**Effective Date:** [DATE]

## 1) Goal
Onboard handymen quickly while maintaining minimum verification and quality controls.

## 2) Onboarding steps
1) Create account and choose role = handyman
2) Complete profile:
   - full name
   - service categories/skills selected
   - location coverage (state/LGA/local council)
   - availability toggle and optional note
3) Verification:
   - address
   - ID type and ID number
   - upload ID image
4) Upload work samples/portfolio (tiles)

## 3) Quality gates
- Verified badge required before accepting jobs: [YES/NO] (decide)
- Minimum portfolio images required: [0/4] (decide)

## 4) First-job monitoring
Monitor:
- time-to-arrival
- cancellation/decline rate
- dispute rate and ratings
